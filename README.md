# Analysis of Gemini 3.1 Pro on Brazilian ICPC Problems

This repository contains the dataset, model outputs, and evaluation logs for the: **Analysis of Gemini 3.1 Pro's Accuracy in Solving Competitive Programming Problems**.

## Dataset
The `data/problems.csv` file contains metadata for 50 advanced algorithmic problems scraped from the Brazilian ICPC Subregionals and Latin American Regionals (2020–2025). There you can find the codes from gemini and submit them for reproducibility.

## Model Outputs
The `data/executions.csv` file contains the raw, unedited zero-shot responses generated by Gemini 3.1 Pro, including:
* The C++14 source code.
* The textual explanation and Big-O complexity analysis.
* The official Online Judge verdict (Accepted, Wrong Answer, or Time Limit Exceeded).

## Prompts

The `data/prompts.csv` file contains the two prompts used over all the 50 problems studied. The first one being the one used to generate the explanation and the code with gemini, and the other one to make gemini propose a problem similar to the one solved.
